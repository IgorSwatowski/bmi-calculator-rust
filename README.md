# BMI Calculator Rust

A body mass index (BMI) calculator written in Rust programming language.

## Introduction

The BMI Calculator Rust is a simple program that calculates your body mass index based on your weight and height. 
The program outputs your BMI score and provides you with information about whether your weight is in a healthy range or not.

## Installation

To run this program, you will need to have the Rust programming language installed on your computer. 
If you do not have Rust installed, you can download it from the official website: https://www.rust-lang.org/.

Clone the repository to your local machine and run the following command in the terminal to build and run the program:

```cargo run```

## Usage

The program will prompt you to enter your weight in kilograms and height in centimeters. 
Once you have entered your information, the program will calculate your BMI and provide you with a result.
